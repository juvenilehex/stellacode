### D-01. Vite Proxy 포트 불일치 (버그)

**현재 상태**: `client/vite.config.ts`의 proxy가 `localhost:3333`을 가리키지만, 서버는 `3001`에서 실행됨.

```

server config   → port: 3001
```

> 현재 동작하는 이유: Vite HMR 프록시를 안 타는 경로로 연결되고 있거나, 환경에 따라 3333을 쓰는 경우가 있을 수 있음. 하지만 클린 환경에서 `npm run dev` 시 API 호출이 실패할 가능성 있음.

**선택지**:
- [ ] A. vite proxy를 3001로 수정 (서버 기본값에 맞춤)
- [ ] B. 서버 기본 포트를 3333으로 변경
- [ ] C. 둘 다 env 변수로 통일 (`STELLA_PORT` 하나로)
