# StellaCode Launch Copy

## Core Message

You don't always need to understand code by reading it.
Sometimes you can understand it by looking at it.

## Slogan

This is not a productivity tool.
It's a way to look at your code and think.

## One-liners (README / site / anywhere)

- A way to see code, not just read it.
- StellaCode is not about writing code. It's about seeing what you wrote.
- You read code line by line. StellaCode lets you see it all at once.
- Git shows history. Static analysis shows structure. StellaCode shows shape.

Title: Show HN: StellaCode -- See your codebase as a constellation

Body:

Sometimes I just want to look at my project and feel like I understand it again.

So I built StellaCode, a tool that turns a codebase into a 3D constellation. Files are stars, imports draw lines between them, and git history shows which files have been moving together over time.

It also notices when AI agents have touched your code -- those files glow a little differently.

npx stellacode

No config. Just point it at a directory and look.

## Positioning

### Identity

StellaCode is not a development tool.
It's a way to see and understand a codebase as a system.

Same family as: why's Poignant Guide, Processing, Observable, Bret Victor projects.
Direction: "a new way to understand code" -- concept project, not dev tool.

### Message Structure

1. Emotion -- "Sometimes I just want to look at my project..."
2. Metaphor -- constellation / stars / observatory
3. Function -- files, imports, git history, AI touched
4. Usage -- npx stellacode
5. Philosophy -- not productivity, think

This reads like: someone had an interesting thought, built a tool from it,
and now we can try it too.

### Words

- NOT: faster, smarter, automated, AI powered, productivity, optimize, scale
- YES: look, understand, constellation, think, observatory, see, shape
- Feel: somewhere between a tool, an artwork, and a thinking instrument

### Tone

- Not grand. Not startup. Not "change the world."
- Quiet. Personal. "I made this because it was interesting."
- Happy if someone finds peace looking at their project.
